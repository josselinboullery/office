# Formation PowerPoint BTS - classement du dossier

Organisation creee pour preparer une formation PowerPoint d'une journee, niveau debutant / faux debutant, pour BTS assistants commerciaux.

Classement effectue le 2026-05-07. Aucun fichier supprime. Aucun contenu PowerPoint, Word, PDF, image, audio ou video modifie. Les noms de fichiers ont ete conserves.

## Structure

- `01_Supports_de_cours` : programmes, fiches pratiques, tutoriels, methodologie.
- `02_Exercices` : fichiers de travail pour les apprenants.
- `03_Corriges` : versions terminees ou probablement corrigees.
- `04_Exemples_et_modeles` : exemples complets, modeles et cas commerciaux.
- `05_Ressources_images_logos` : images, logos, GIF, audio et video.
- `06_Archives_a_verifier` : fichiers hors cible, doublons apparents, exports web, brouillons, administratif.
- `07_Support_final_a_creer` : emplacement vide reserve au futur support final.

## Fichiers les plus utiles

- `01_Supports_de_cours\Programmes\PROGRAMME PowerPoint initiation complet.docx` : base principale pour le deroule de la journee.
- `01_Supports_de_cours\Programmes\formation_powerpoint-initiation.docx` : source complementaire initiation.
- `01_Supports_de_cours\Design_et_methodologie\Élaboration d’un diaporama.pptx` : support sur les regles de conception.
- `01_Supports_de_cours\Design_et_methodologie\quelques conseils utiles.pptx` : conseils presentation/design.
- `02_Exercices\Initiation\Exercice Pratique Init.pptx` : exercice court de demarrage.
- `02_Exercices\Recette_crepes\Recette des crêpes brut.pptx` : base brute pour exercice guide.
- `02_Exercices\Recette_crepes\Recette des crêpes sans mise en forme.pptx` : bon exercice de mise en forme.
- `03_Corriges\Recette_crepes\Recette des crêpes.pptx` et `03_Corriges\Recette_crepes\crèpes.pptx` : corriges ou exemples aboutis.

## Fichiers a verifier manuellement

- `06_Archives_a_verifier\Brouillons_douteux` : fichiers tests, noms non explicites, fichier `.kra`.
- `06_Archives_a_verifier\Niveau_intermediaire_perfectionnement` : contenus utiles possibles, mais niveau trop avance pour la cible principale.
- `06_Archives_a_verifier\Exports_web` : exports web et caches telecharges, a nettoyer si reutilisation.
- `06_Archives_a_verifier\Administratif_Hermes` : documents administratifs, hors support de cours.
- `06_Archives_a_verifier\Administratif_Hermes\feuilles de présences hermes\Incruster une vidéo derrière du texte.pdf` : doublon apparent avec le PDF range dans les tutoriels.

## Prochaines etapes

1. Construire le deroule journee depuis `PROGRAMME PowerPoint initiation complet.docx`.
2. Selectionner 3 ou 4 blocs pratiques : prise en main, mise en forme, images/SmartArt/tableau, diaporama final.
3. Utiliser l'exercice recette de crepes comme fil rouge.
4. Creer le support final dans `07_Support_final_a_creer`.
5. Produire une version apprenant et une version formateur/corrige.
