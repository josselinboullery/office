# Exercices PowerPoint - BTS assistants commerciaux

Ces exercices utilisent les fichiers déjà présents dans le dossier. Ils sont progressifs et peuvent être réalisés pendant la journée de formation.

## Vue d’ensemble

| Exercice | Durée | Objectif principal | Fichiers |
|---|---:|---|---|
| 1. Remettre en forme une diapositive | 30 min | Lisibilité, hiérarchie, mise en page | `Recette des crêpes sans mise en forme.pptx` |
| 2. Créer une présentation commerciale courte | 40 min | Structurer une offre commerciale | `GPF1_2022_BILLETTERIE_PREMIUM.html`, ressources images |

## Exercice 1 - Remettre en forme une diapositive mal présentée

Durée estimée : 30 minutes.  
Moment conseillé : fin de matinée.

### Fichiers nécessaires

- `02_Exercices\Recette_crepes\Recette des crêpes sans mise en forme.pptx`
- `03_Corriges\Recette_crepes\Recette des crêpes.pptx`
- `03_Corriges\Recette_crepes\crèpes.pptx`

### Objectif pédagogique

Être capable d’améliorer une diapositive trop brute pour la rendre plus lisible, plus structurée et plus professionnelle.

### Consigne apprenant

Ouvrez `Recette des crêpes sans mise en forme.pptx`.

Choisissez une diapositive parmi :

- `Ustensiles`
- `Ingrédients`
- `Méthodologie`
- `Exemples de présentation`

Votre mission :

1. Garder le contenu de départ.
2. Donner un titre clair à la diapositive.
3. Réduire ou organiser le texte si nécessaire.
4. Aligner les objets.
5. Améliorer la lisibilité : taille, contraste, espace.
6. Ajouter une forme ou une image uniquement si cela aide à comprendre.
7. Enregistrer votre fichier sous `exercice_1_prenom_nom.pptx`.

### Critères de réussite

- le titre indique clairement l’idée de la diapositive ;
- le contenu est lisible à distance ;
- les éléments sont alignés ;
- la diapositive n’est pas surchargée ;
- le style reste sobre ;
- les images ne sont pas déformées.

### Corrigé ou correction formateur

Comparer avec les fichiers corrigés :

- `03_Corriges\Recette_crepes\Recette des crêpes.pptx`
- `03_Corriges\Recette_crepes\crèpes.pptx`

Questions de correction :

- Quelle information regarde-t-on en premier ?
- Le titre aide-t-il à comprendre ?
- Peut-on lire la diapositive depuis le fond de la salle ?
- Y a-t-il trop d’objets ?
- Les alignements sont-ils réguliers ?

Grille rapide sur 10 points :

| Critère | Points |
|---|---:|
| Titre clair | 2 |
| Texte réduit et hiérarchisé | 2 |
| Alignement et espace | 2 |
| Contraste et lisibilité | 2 |
| Sobriété générale | 2 |

## Exercice 2 - Créer une présentation commerciale courte

Durée estimée : 40 minutes.  
Moment conseillé : après les notions de structure et avant les objets avancés.

### Fichiers nécessaires

- `04_Exemples_et_modeles\Exemples_commerciaux\GPF1_2022_BILLETTERIE_PREMIUM.html`
- `04_Exemples_et_modeles\Exemples_commerciaux\GPF1_2022_BILLETTERIE_PREMIUM_files`
- `05_Ressources_images_logos\Hermes_logos_images`
- `05_Ressources_images_logos\Images album`

Note : les images de marque sont utilisées ici comme ressources pédagogiques internes. Pour un usage externe, vérifier les droits.

### Objectif pédagogique

Structurer une courte présentation commerciale avec un message clair, une proposition de valeur et une conclusion orientée action.

### Consigne apprenant

Créez une présentation de 4 diapositives pour présenter une offre commerciale fictive.

Vous pouvez choisir un des thèmes suivants :

- une offre événementielle premium inspirée du dossier `GPF1_2022_BILLETTERIE_PREMIUM` ;
- une offre de produit haut de gamme à partir des images du dossier `Hermes_logos_images` ;
- une offre de service simple pour une entreprise locale.

Structure obligatoire :

1. Diapositive 1 : titre de l’offre + accroche.
2. Diapositive 2 : besoin ou problème du client.
3. Diapositive 3 : solution proposée + 3 bénéfices.
4. Diapositive 4 : prochaine étape ou appel à l’action.

Contraintes :

- 1 idée principale par diapositive ;
- pas plus de 5 lignes de texte par diapositive ;
- au moins 1 image ;
- au moins 1 forme ou pictogramme ;
- style cohérent sur les 4 diapositives ;
- enregistrer sous `exercice_2_presentation_commerciale_prenom_nom.pptx`.

### Critères de réussite

- l’offre est compréhensible ;
- le besoin client est clair ;
- les bénéfices sont formulés simplement ;
- la présentation donne envie d’écouter ;
- la conclusion propose une action concrète ;
- le style est cohérent.

### Corrigé ou grille de correction

Il n’existe pas de corrigé unique. Le formateur corrige avec cette grille.

| Critère | Points |
|---|---:|
| Structure en 4 diapositives respectée | 2 |
| Besoin client clairement formulé | 2 |
| Arguments de vente concrets | 2 |
| Mise en page lisible et professionnelle | 2 |
| Cohérence graphique | 1 |
| Conclusion orientée action | 1 |

Correction orale :

- demander à 2 ou 3 participants de présenter en 1 minute ;
- faire identifier par le groupe le message principal ;
- proposer une amélioration précise par présentation.

## Exercice 3 - Image, tableau et arguments de vente

Durée estimée : 40 minutes.  
Moment conseillé : après la démonstration images, tableaux et graphiques.

### Fichiers nécessaires

- `02_Exercices\Donnees_exercices\données tableau crêpes.xlsx`
- `02_Exercices\Recette_crepes\Recette des crêpes brut.pptx`
- `05_Ressources_images_logos\Images_crepes\images`
- `03_Corriges\Recette_crepes\Recette des crêpes.pptx`

### Objectif pédagogique

Créer une diapositive commerciale qui combine image, tableau ou graphique, et argument de vente.

### Consigne apprenant

Créez 3 diapositives sur le thème d’une offre fictive : “Atelier crêpes pour événement client”.

Diapositive 1 : présentation de l’offre

- titre clair ;
- une image de crêpe ;
- une phrase d’accroche.

Diapositive 2 : ingrédients ou organisation

- tableau court à partir du fichier Excel ;
- maximum 4 ou 5 lignes ;
- ligne d’en-tête visible ;
- titre explicite.

Diapositive 3 : argument commercial

- 3 arguments de vente ;
- une forme ou un SmartArt simple ;
- une conclusion courte.

Option si le groupe avance vite :

- ajouter un graphique simple à partir des données du fichier Excel ;
- donner au graphique un titre qui explique l’idée.

### Critères de réussite

- les images sont nettes et non déformées ;
- le tableau est lisible ;
- les arguments sont orientés client ;
- les objets sont alignés ;
- la présentation reste sobre ;
- chaque diapositive a un rôle clair.

### Corrigé ou correction formateur

S’appuyer sur :

- `03_Corriges\Recette_crepes\Recette des crêpes.pptx`
- `03_Corriges\Recette_crepes\crèpes.pptx`

Exemple d’arguments attendus :

- animation conviviale pour un événement client ;
- format simple à comprendre ;
- visuels chaleureux et mémorisables ;
- possibilité de présenter une offre sous forme d’étapes.

Grille rapide sur 15 points :

| Critère | Points |
|---|---:|
| Diapositive 1 claire et attractive | 3 |
| Tableau lisible et bien présenté | 3 |
| Arguments orientés client | 3 |
| Utilisation correcte image / forme / SmartArt | 3 |
| Cohérence visuelle entre les diapositives | 3 |

## Exercice 4 - Exercice final : mini-présentation professionnelle

Durée estimée : 55 minutes.  
Moment conseillé : fin d’après-midi.

### Fichiers nécessaires

Selon le thème choisi :

- `05_Ressources_images_logos\Images_crepes\images`
- `05_Ressources_images_logos\Hermes_logos_images`
- `05_Ressources_images_logos\Images album`
- `02_Exercices\Donnees_exercices\données tableau crêpes.xlsx`
- `04_Exemples_et_modeles\Exemples_commerciaux\GPF1_2022_BILLETTERIE_PREMIUM.html`
- `01_Supports_de_cours\Fiches_pratiques\Les raccourcis claviers.docx`

### Objectif pédagogique

Créer une présentation complète de 5 à 6 diapositives, structurée, lisible, cohérente et adaptée à un contexte commercial.

### Consigne apprenant

Créez une mini-présentation professionnelle de 5 à 6 diapositives.

Sujet :

- une offre commerciale ;
- un produit ;
- un service ;
- un événement client ;
- ou un mini-bilan commercial fictif.

Structure recommandée :

1. Titre + accroche.
2. Contexte ou besoin client.
3. Proposition ou solution.
4. Arguments / bénéfices.
5. Donnée, tableau, graphique ou preuve.
6. Conclusion + prochaine étape.

Contraintes :

- 5 à 6 diapositives ;
- une idée principale par diapositive ;
- au moins une image ;
- au moins un tableau ou graphique ;
- au moins une forme, icône ou SmartArt ;
- un thème ou une cohérence de couleurs ;
- transitions sobres ;
- export PDF ;
- présentation orale de 2 minutes.

Fichiers à rendre :

- `exercice_final_prenom_nom.pptx`
- `exercice_final_prenom_nom.pdf`

### Critères de réussite

- le message commercial est clair ;
- la structure aide à suivre le raisonnement ;
- les textes sont courts ;
- la présentation est lisible ;
- les objets sont alignés ;
- les images sont bien utilisées ;
- le style est cohérent ;
- l’export PDF est vérifié ;
- la prise de parole s’appuie sur les diapositives sans les lire mot à mot.

### Corrigé ou grille de correction

Utiliser le fichier :

- `grille_correction_exercice_final.md`

Correction conseillée :

1. Chaque participant présente en 2 minutes.
2. Le formateur évalue avec la grille.
3. Le groupe formule un point fort et une amélioration.
4. Le participant note une modification à faire avant usage réel.

## Variante courte si le groupe est lent

Réduire l’exercice final à 4 diapositives :

1. Titre.
2. Besoin client.
3. Solution + bénéfices.
4. Conclusion + action.

Supprimer l’obligation de graphique, mais garder au moins une image et une cohérence graphique.

## Variante avancée si le groupe est à l’aise

Ajouter :

- une diapositive construite avec le masque ;
- une animation d’apparition sur les arguments ;
- un lien hypertexte ou bouton de navigation ;
- une version PDF prête à envoyer à un client.
