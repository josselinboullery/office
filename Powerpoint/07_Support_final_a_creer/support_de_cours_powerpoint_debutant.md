# Support de cours - PowerPoint débutant

Public : BTS assistants commerciaux.  
Durée prévue : 1 journée.  
Positionnement : débutant / faux débutant.  
But : savoir créer une présentation courte, claire et professionnelle dans un contexte commercial.

## Sources utilisées

Ce support s’appuie sur les fichiers disponibles dans le dossier organisé :

- `Élaboration d’un diaporama.pptx` pour les règles de conception ;
- `quelques conseils utiles.pptx` pour les exemples de présentation visuelle ;
- `PROGRAMME PowerPoint initiation complet.docx` pour la progression ;
- `formation_powerpoint-initiation.docx` pour les objectifs de base ;
- `Les raccourcis claviers.docx` pour les manipulations rapides ;
- `LE MASQUE DES DIAPOSITIVES.docx` pour la cohérence graphique ;
- `Exercice Pratique Init.pptx` et l’exercice recette de crêpes pour les mises en pratique.

## 1. À quoi sert une présentation PowerPoint ?

Une présentation PowerPoint n’est pas un document Word projeté.

Elle sert à :

- soutenir l’attention du public ;
- aider à comprendre et mémoriser ;
- structurer un discours ;
- rendre une idée plus visible ;
- accompagner une prise de parole.

Dans un contexte commercial, elle peut servir à :

- présenter une offre ;
- expliquer un produit ou un service ;
- valoriser des bénéfices client ;
- comparer deux solutions ;
- préparer un rendez-vous client ;
- présenter un bilan ou une action commerciale.

Règle simple : le public doit comprendre l’idée principale de la diapositive en quelques secondes.

## 2. Les 5 règles d’or d’une diapositive efficace

Ces règles viennent du support `Élaboration d’un diaporama.pptx`.

1. Une diapositive = une idée principale.
2. Le public doit se concentrer sur le présentateur, pas seulement sur l’écran.
3. Les objets doivent avoir une taille adaptée.
4. Le contraste doit permettre une lecture facile.
5. Le nombre d’objets doit rester limité.

Application concrète :

- remplacer les paragraphes par des mots-clés ;
- utiliser un titre explicite ;
- garder de l’espace vide ;
- éviter les couleurs trop proches ;
- supprimer les éléments qui ne servent pas le message.

## 3. Préparer avant de créer

Avant d’ouvrir PowerPoint ou avant de décorer les diapositives, préparer le message.

Méthode simple :

1. Définir le public : client, collègue, responsable, jury.
2. Définir l’objectif : informer, convaincre, comparer, vendre, présenter.
3. Écrire le message principal en une phrase.
4. Faire un plan court.
5. Prévoir une diapositive par idée.

Exemple de plan commercial en 5 diapositives :

1. Titre de l’offre.
2. Besoin ou problème du client.
3. Solution proposée.
4. Bénéfices et preuves.
5. Prochaine étape.

Un bon plan évite deux erreurs fréquentes :

- mettre trop d’informations sur une seule diapositive ;
- créer une présentation jolie mais confuse.

## 4. Se repérer dans PowerPoint

Les onglets à connaître :

- `Fichier` : créer, ouvrir, enregistrer, exporter.
- `Accueil` : texte, paragraphes, disposition, nouvelles diapositives.
- `Insertion` : images, tableaux, graphiques, formes, zones de texte.
- `Création` : thèmes, couleurs, arrière-plan.
- `Transitions` : passage entre diapositives.
- `Animations` : apparition ou mouvement d’objets.
- `Diaporama` : lancer et configurer la présentation.
- `Affichage` : modes d’affichage, règle, repères, masque.

Point important : si le fichier s’appelle encore `Présentation1`, il n’est probablement pas enregistré avec un vrai nom. Il y a un risque de perte.

## 5. Créer et gérer les diapositives

Actions de base :

- créer une nouvelle diapositive ;
- choisir une disposition ;
- déplacer une diapositive ;
- dupliquer une diapositive ;
- supprimer une diapositive ;
- passer en mode trieuse de diapositives pour voir l’ensemble.

Dispositions utiles :

- `Titre` : page d’ouverture ;
- `Titre et contenu` : usage général ;
- `Deux contenus` ou `Comparaison` : comparer deux offres ;
- `Section` : annoncer une nouvelle partie ;
- `Vide` : construire une diapositive sur mesure.

Conseil formateur : faire utiliser les dispositions avant les zones de texte libres. Les débutants gagnent en structure.

## 6. Titres, texte et listes

Le titre doit annoncer l’idée de la diapositive.

Titre faible :

- `Introduction`
- `Produit`
- `Conclusion`

Titre plus utile :

- `Notre offre répond au besoin de rapidité du client`
- `Trois arguments pour choisir la formule premium`
- `Prochaine étape : confirmer le rendez-vous`

Pour le texte :

- écrire court ;
- utiliser des mots-clés ;
- limiter les puces ;
- éviter les phrases complètes quand elles ne sont pas nécessaires ;
- utiliser une taille lisible.

Raccourcis utiles issus de `Les raccourcis claviers.docx` :

- `TAB` : augmenter le niveau d’une liste ;
- `MAJ + TAB` : réduire le niveau ;
- `CTRL + C` : copier ;
- `CTRL + V` : coller ;
- `CTRL + X` : couper ;
- `CTRL + A` : tout sélectionner selon le contexte.

## 7. Hiérarchie visuelle

La hiérarchie visuelle aide le public à savoir quoi regarder d’abord.

Pour créer une hiérarchie :

- grand titre ;
- sous-titre ou idée clé ;
- éléments secondaires plus petits ;
- couleurs limitées ;
- alignements nets ;
- espace autour des objets.

Exemple :

- titre : “Une offre simple pour gagner du temps”
- trois arguments : “Rapide”, “Clair”, “Rentable”
- une image ou icône pour illustrer.

Erreur courante : tout mettre à la même taille. Si tout est important, plus rien ne ressort.

## 8. Mise en page

Une mise en page propre repose sur quelques règles simples :

- aligner les objets ;
- garder des marges ;
- éviter les objets collés aux bords ;
- équilibrer texte et image ;
- utiliser l’espace vide ;
- répéter les mêmes placements d’une diapositive à l’autre.

Méthode rapide :

1. Placer le titre en haut.
2. Définir une zone de contenu.
3. Aligner les objets.
4. Vérifier les tailles.
5. Supprimer ce qui ne sert pas.

Dans une présentation commerciale, la mise en page doit aider à comparer, décider ou retenir un argument.

## 9. Images, icônes et formes

Les images doivent renforcer le message.

Règles :

- choisir une image nette ;
- éviter de déformer l’image ;
- rogner plutôt que compresser ;
- garder les proportions ;
- ne pas utiliser trop d’images sur une même diapositive ;
- vérifier les droits si la présentation sort du cadre pédagogique.

Fichiers utiles du dossier :

- `05_Ressources_images_logos\Images_crepes\images`
- `05_Ressources_images_logos\Images album`
- `05_Ressources_images_logos\Hermes_logos_images`

Formes :

- utiles pour encadrer une idée ;
- utiles pour créer un bouton, un repère ou un schéma simple ;
- à garder sobres.

Raccourcis utiles :

- maintenir `MAJ` pour garder les proportions ;
- maintenir `CTRL` pour dupliquer ;
- sélectionner plusieurs objets avec `MAJ + clic`.

## 10. Tableaux et graphiques

Un tableau sert à organiser une information.

Bon tableau :

- peu de colonnes ;
- titre clair ;
- ligne d’en-tête visible ;
- alignement propre ;
- couleurs sobres ;
- données lisibles à distance.

Un graphique sert à montrer une tendance ou une comparaison.

Bon graphique :

- titre qui explique l’idée ;
- peu de séries ;
- couleurs cohérentes ;
- légende lisible ;
- pas d’effet 3D inutile.

Fichier utile :

- `02_Exercices\Donnees_exercices\données tableau crêpes.xlsx`

Exemple d’utilisation :

- créer un tableau d’ingrédients ;
- créer un graphique simple sur la quantité de crêpes ;
- ajouter un titre qui interprète la donnée.

## 11. SmartArt

Un SmartArt permet de transformer une liste en schéma.

Usages simples :

- étapes d’un processus ;
- hiérarchie ;
- cycle ;
- comparaison ;
- relation entre idées.

Conseil issu du programme existant : utiliser le volet texte du SmartArt. Il permet de modifier le contenu sans casser la mise en forme.

Exemple :

Méthodologie de la recette :

1. Mélanger les ingrédients secs.
2. Incorporer les œufs et le beurre.
3. Diluer avec le lait.
4. Cuire à la poêle.

Cette liste peut devenir un SmartArt de processus.

## 12. Alignement, distribution et groupement

Aligner signifie placer les objets sur une même ligne.

Distribuer signifie répartir les objets avec le même espace entre eux.

Grouper signifie réunir plusieurs objets pour les déplacer ensemble.

Usages :

- aligner trois images ;
- répartir trois arguments ;
- grouper une icône et son texte ;
- déplacer un bloc complet sans casser la mise en page.

Point de vigilance : bien choisir si l’alignement se fait par rapport aux objets ou par rapport à la diapositive.

## 13. Thèmes et cohérence graphique

Un thème donne une cohérence :

- couleurs ;
- polices ;
- arrière-plan ;
- styles.

Pour une formation débutant, l’objectif n’est pas de créer un thème complexe. L’objectif est d’obtenir une présentation uniforme.

Bonnes pratiques :

- choisir un thème simple ;
- garder 2 ou 3 couleurs principales ;
- utiliser les mêmes polices ;
- répéter les mêmes placements ;
- éviter de changer de style à chaque diapositive.

## 14. Masque des diapositives

Le masque sert à contrôler la structure commune d’une présentation.

Source :

- `LE MASQUE DES DIAPOSITIVES.docx`

À retenir :

- le masque se trouve dans l’onglet `Affichage` ;
- une disposition mère transmet ses caractéristiques aux dispositions filles ;
- les espaces réservés permettent de créer des zones faciles à remplir ;
- le masque aide à gérer les couleurs et polices de la présentation.

Pour des débutants, utiliser le masque seulement pour comprendre la cohérence. Ne pas en faire un objectif technique avancé.

## 15. Transitions et animations

Une transition agit entre deux diapositives.

Une animation agit sur un objet dans une diapositive.

Bon usage :

- utiliser peu d’effets ;
- garder le même style ;
- choisir des animations simples ;
- animer pour révéler une information au bon moment ;
- éviter les effets spectaculaires.

À éviter :

- animations différentes partout ;
- texte qui arrive lettre par lettre ;
- sons inutiles ;
- transitions trop longues.

Exemple utile :

- faire apparaître trois arguments un par un pendant l’explication orale.

## 16. Préparer la présentation orale

PowerPoint accompagne la prise de parole.

Préparation :

- relire les titres ;
- vérifier que chaque diapositive a une idée ;
- ajouter des notes si nécessaire ;
- lancer le diaporama ;
- chronométrer ;
- prévoir les transitions orales.

Pour une présentation commerciale courte :

1. Annoncer le besoin.
2. Présenter la solution.
3. Donner 2 ou 3 arguments.
4. Montrer une preuve ou un exemple.
5. Conclure avec une action claire.

## 17. Export PDF

Exporter en PDF permet :

- d’envoyer une version stable ;
- de préserver la mise en page ;
- de partager sans modifier le fichier source.

Étapes :

1. `Fichier`
2. `Exporter` ou `Enregistrer sous`
3. choisir `PDF`
4. vérifier le nom du fichier
5. ouvrir le PDF pour contrôler le résultat

Point de vigilance : toujours relire le PDF exporté. Un élément peut être coupé, trop petit ou mal placé.

## 18. Checklist avant rendu

Structure :

- la présentation a un début, un développement et une fin ;
- chaque diapositive a une idée principale ;
- les titres sont explicites.

Lisibilité :

- le texte est assez grand ;
- le contraste est suffisant ;
- les tableaux et graphiques sont lisibles.

Mise en page :

- les objets sont alignés ;
- les marges sont respectées ;
- le style est cohérent.

Technique :

- le fichier est enregistré ;
- les images ne sont pas déformées ;
- les animations restent sobres ;
- le PDF est exporté et vérifié.

Message commercial :

- le besoin client est compréhensible ;
- les arguments sont clairs ;
- la conclusion indique une prochaine étape.
