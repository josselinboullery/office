# Déroulé pédagogique - PowerPoint débutant - 1 jour

Public : BTS assistants commerciaux, niveau débutant ou faux débutant.  
Durée : 1 journée, 7 heures de formation hors pause déjeuner.  
Objectif général : créer une présentation PowerPoint simple, structurée, lisible et professionnelle, adaptée à un contexte commercial.

## Séquence 1 - Accueil et cadrage

Horaire : 09:00 - 09:20

Objectifs :

- comprendre le niveau réel des participants ;
- clarifier le résultat attendu : une présentation courte, claire, professionnelle ;
- installer une logique pratique : chaque notion est suivie d’une manipulation.

Déroulé :

1. Présentation rapide du formateur et du programme.
2. Tour de table : “Dans quel contexte commercial pourriez-vous utiliser PowerPoint ?”
3. Question de départ : “Qu’est-ce qui rend une présentation difficile à lire ?”
4. Annonce du fil rouge : améliorer puis créer une présentation courte à partir d’un cas simple.

Points de vigilance :

- rassurer les débutants ;
- éviter les démonstrations trop longues ;
- partir de situations commerciales concrètes : offre, bilan, argumentaire, rendez-vous client.

## Séquence 2 - Rôle d’une présentation et règles d’or

Horaire : 09:20 - 09:45

Sources :

- `Élaboration d’un diaporama.pptx`
- `quelques conseils utiles.pptx`

Objectifs :

- comprendre qu’une présentation soutient un message, pas un document complet ;
- appliquer les règles de base : une idée par diapositive, peu de texte, contraste, cohérence.

Notions abordées :

- rôle d’une présentation : soutenir l’attention, aider à mémoriser, structurer le discours ;
- une diapositive = une idée principale ;
- le public écoute le présentateur, il ne lit pas un dossier ;
- une image peut remplacer un long paragraphe ;
- limiter le nombre d’objets par diapositive.

Démonstration :

- montrer quelques diapositives de `Élaboration d’un diaporama.pptx` ;
- faire commenter les règles d’or ;
- montrer dans `quelques conseils utiles.pptx` les exemples : grande image, citation courte, infographie, tableau.

Mini-activité :

- demander aux participants de reformuler une mauvaise diapositive imaginaire en une seule phrase clé.

Correction :

- faire ressortir les critères : message unique, titre clair, texte réduit, contraste, alignement.

Points de vigilance :

- éviter le vocabulaire “design” trop abstrait ;
- traduire chaque règle en geste PowerPoint concret.

## Séquence 3 - Interface PowerPoint et gestion des fichiers

Horaire : 09:45 - 10:20

Sources :

- `PROGRAMME PowerPoint initiation complet.docx`
- `formation_powerpoint-initiation.docx`

Objectifs :

- se repérer dans PowerPoint ;
- savoir créer, ouvrir, enregistrer et exporter une présentation.

Notions abordées :

- onglet `Fichier` : créer, ouvrir, enregistrer, exporter ;
- onglet `Accueil` : texte, paragraphes, nouvelles diapositives ;
- onglet `Insertion` : tableaux, images, formes, graphiques ;
- onglet `Création` : thèmes et variantes ;
- onglet `Transitions` : passage entre diapositives ;
- onglet `Animations` : animation d’objets ;
- onglet `Diaporama` : lancer la présentation ;
- onglet `Affichage` : modes d’affichage et masque.

Démonstration :

1. Créer une présentation vierge.
2. Enregistrer immédiatement le fichier.
3. Montrer la différence entre un fichier non enregistré et un fichier nommé.
4. Enregistrer au format `.pptx`.
5. Exporter rapidement en PDF pour montrer la finalité.

Mini-pratique :

- chaque participant crée un fichier nommé `presentation_prenom_nom.pptx`.

Points de vigilance :

- rappeler que `Présentation1` indique un fichier non enregistré ;
- insister sur l’enregistrement régulier ;
- ne pas détailler tous les onglets : seulement ceux utilisés pendant la journée.

## Séquence 4 - Créer et structurer une présentation

Horaire : 10:30 - 11:10

Sources :

- `PROGRAMME PowerPoint initiation complet.docx`
- `Élaboration d’un diaporama.pptx`

Objectifs :

- préparer une présentation avant de la décorer ;
- choisir les bonnes dispositions de diapositives ;
- structurer un message commercial simple.

Notions abordées :

- faire un plan avant d’ouvrir trop d’outils ;
- une grande idée = une diapositive ;
- utiliser les dispositions : titre, titre et contenu, comparaison, section ;
- créer, dupliquer, déplacer et supprimer une diapositive ;
- mode plan : titres et idées secondaires.

Démonstration :

Créer une structure en 5 diapositives :

1. Titre de l’offre.
2. Contexte / besoin client.
3. Solution proposée.
4. Bénéfices / arguments.
5. Prochaine étape / contact.

Mini-pratique :

- les participants créent les 5 diapositives vides avec un titre clair sur chacune.

Correction :

- vérifier que chaque titre annonce le contenu ;
- corriger les titres trop vagues comme “Introduction”, “Partie 1”, “Conclusion”.

Points de vigilance :

- un titre doit aider le public à comprendre, pas seulement nommer une partie ;
- éviter de commencer par les couleurs ou les images.

## Séquence 5 - Texte, titres, listes et tableaux

Horaire : 11:10 - 12:00

Sources :

- `Exercice Pratique Init.pptx`
- `Les raccourcis claviers.docx`
- `Recette des crêpes brut.pptx`

Objectifs :

- écrire peu mais utile ;
- hiérarchiser les listes ;
- créer un tableau lisible.

Notions abordées :

- différence entre titre et zone de texte ;
- listes à puces ;
- hiérarchie avec `TAB` et `MAJ + TAB` ;
- taille de police ;
- alignement du texte ;
- création d’un tableau ;
- fusionner, fractionner, bordures, couleurs simples.

Démonstration :

1. Ouvrir `Exercice Pratique Init.pptx`.
2. Montrer les consignes : zone de texte, tableau, graphique, images, formes, SmartArt.
3. Dans un nouveau fichier, créer une diapositive avec une liste hiérarchisée.
4. Créer un tableau simple à partir des ingrédients de la recette de crêpes.

Pratique guidée :

- créer une diapositive “Ingrédients” avec un tableau propre ;
- utiliser une ligne d’en-tête ;
- éviter les bordures trop lourdes.

Correction :

- 2 ou 3 exemples projetés ;
- correction collective sur lisibilité, taille, alignement.

Points de vigilance :

- ne pas transformer la diapositive en document Word ;
- éviter les tableaux trop petits ;
- préférer 3 à 5 lignes utiles à un tableau complet illisible.

## Séquence 6 - Exercice 1 : remettre en forme une diapositive

Horaire : 12:00 - 12:30

Fichier :

- `02_Exercices\Recette_crepes\Recette des crêpes sans mise en forme.pptx`

Objectif :

- transformer une diapositive peu lisible en diapositive claire.

Consigne rapide :

- choisir une diapositive de la recette ;
- améliorer titre, texte, alignement, espace, contraste ;
- garder le contenu, mais rendre la diapositive plus professionnelle.

Correction :

- comparer avec `03_Corriges\Recette_crepes\Recette des crêpes.pptx` ;
- verbaliser les choix : “j’ai réduit”, “j’ai aligné”, “j’ai regroupé”, “j’ai rendu le titre plus explicite”.

Points de vigilance :

- les participants veulent souvent ajouter des effets ; rappeler que l’objectif est la lisibilité.

## Séquence 7 - Images, formes, icônes et alignements

Horaire : 13:30 - 14:20

Sources :

- `05_Ressources_images_logos\Images_crepes\images`
- `05_Ressources_images_logos\Hermes_logos_images`
- `Les raccourcis claviers.docx`

Objectifs :

- insérer et ajuster une image ;
- utiliser formes et icônes comme supports de message ;
- aligner et distribuer proprement des objets.

Notions abordées :

- insérer une image ;
- rogner une image ;
- conserver les proportions ;
- mettre une image au bon plan ;
- sélectionner plusieurs objets ;
- aligner, distribuer, grouper ;
- utiliser `MAJ` pour garder les proportions ;
- utiliser `CTRL` pour dupliquer.

Démonstration :

1. Insérer 3 images de crêpes.
2. Les rogner dans des formats différents.
3. Les aligner et les distribuer.
4. Ajouter une forme simple pour mettre en avant un argument.
5. Grouper certains éléments.

Pratique :

- créer une diapositive visuelle “Exemples de présentation” à partir des images disponibles.

Correction :

- vérifier marges, alignement, équilibre, absence de surcharge.

Points de vigilance :

- ne pas déformer les images ;
- éviter les images trop petites ;
- garder assez d’espace vide.

## Séquence 8 - Tableau, graphique et arguments de vente

Horaire : 14:20 - 15:00

Sources :

- `02_Exercices\Donnees_exercices\données tableau crêpes.xlsx`
- `02_Exercices\Recette_crepes\Recette des crêpes brut.pptx`
- `03_Corriges\Recette_crepes\crèpes.pptx`

Objectifs :

- transformer des données en message ;
- créer un tableau ou graphique simple ;
- relier une donnée à un argument.

Notions abordées :

- tableau dans PowerPoint ;
- graphique depuis PowerPoint ou depuis Excel ;
- titre de graphique ;
- choix d’un message commercial ;
- simplification des données.

Démonstration :

- ouvrir le fichier Excel ;
- repérer les colonnes liées au tableau et au graphique ;
- créer un graphique simple “prise de poids en fonction de la quantité de crêpes” ;
- expliquer que le graphique sert à faire passer une idée, pas à montrer toutes les données.

Pratique :

- créer une diapositive avec un tableau court ;
- créer une diapositive avec un graphique ou un indicateur visuel ;
- ajouter un titre qui interprète la donnée.

Correction :

- vérifier que le titre explique l’idée ;
- vérifier que la donnée reste lisible à distance.

Points de vigilance :

- éviter les graphiques 3D ;
- éviter trop de couleurs ;
- supprimer les éléments inutiles si le graphique est chargé.

## Séquence 9 - Thèmes, cohérence, transitions, animations, export

Horaire : 15:15 - 15:45

Sources :

- `LE MASQUE DES DIAPOSITIVES.docx`
- `Exemple Morphose.pptx`
- `PROGRAMME PowerPoint initiation complet.docx`

Objectifs :

- uniformiser une présentation ;
- utiliser thèmes et animations avec modération ;
- préparer l’export PDF.

Notions abordées :

- thème prédéfini ;
- jeu de couleurs ;
- polices cohérentes ;
- principe du masque de diapositives ;
- transitions simples ;
- animations d’apparition ;
- notes du présentateur ;
- export PDF.

Démonstration :

1. Appliquer un thème simple.
2. Modifier une variante de couleur.
3. Montrer le masque sans entrer dans un usage expert.
4. Ajouter une transition sobre.
5. Montrer une animation d’apparition.
6. Exporter en PDF.

Points de vigilance :

- le masque est présenté comme outil de cohérence, pas comme objectif technique principal ;
- les animations doivent aider la compréhension, pas décorer ;
- une transition identique et sobre suffit souvent.

## Séquence 10 - Exercice final et restitution

Horaire : 15:45 - 17:00

Objectif :

- créer une mini-présentation commerciale de 5 à 6 diapositives.

Consigne synthétique :

- choisir une offre ou un produit ;
- construire un message clair ;
- utiliser au moins une image, un tableau ou graphique, des formes simples ;
- appliquer une cohérence graphique ;
- préparer une présentation orale de 2 minutes ;
- exporter en PDF.

Organisation :

- 45 minutes de production ;
- 10 minutes d’export et préparation orale ;
- 15 minutes de restitution et correction.

Correction :

- utiliser `grille_correction_exercice_final.md`.

Points de vigilance :

- la production doit rester courte ;
- le formateur passe dans les rangs pour éviter la surcharge ;
- priorité : structure, lisibilité, cohérence.

## Bilan de fin de journée

Questions à poser :

- Quelle règle allez-vous appliquer en priorité dans vos prochaines présentations ?
- Quel outil PowerPoint vous semble le plus utile pour un contexte commercial ?
- Quelle erreur allez-vous éviter ?

Livrables attendus :

- fichier PowerPoint final ;
- export PDF ;
- mini-présentation orale de 2 minutes.
