# Support de cours — Maîtriser Microsoft Word

## Public visé

Ce support s’adresse à des apprenants qui savent utiliser un ordinateur, mais qui souhaitent apprendre à produire des documents Word propres, structurés, professionnels et faciles à modifier.

Il peut être utilisé pour une formation Word **initiation**, **intermédiaire** ou **perfectionnement**, selon le rythme choisi.

## Version de référence

Ce support prend comme référence **Microsoft Word version bureau**. Word pour le web permet de créer et modifier des documents, mais certaines fonctions avancées sont plus limitées, notamment sur la personnalisation complète des styles, les documents longs, les modèles et certaines fonctionnalités avancées de mise en page. Microsoft propose une comparaison officielle entre Word pour le web et Word version bureau. ([Support Microsoft][1])

---

# Objectif général de la formation

À la fin de la formation, l’apprenant doit être capable de créer, corriger, structurer, mettre en page, illustrer, automatiser, partager et finaliser un document Word professionnel.

L’objectif n’est pas seulement de “savoir mettre du texte en gras”, mais de comprendre la logique de Word :

**un bon document Word est un document structuré avant d’être décoré.**

# Module 1 — Comprendre Word

## Objectif du module

Comprendre à quoi sert Word, comment il fonctionne et pourquoi certains documents deviennent difficiles à modifier.

## 1.1 Word n’est pas seulement une page blanche

Word est un logiciel de traitement de texte. Il permet de créer des courriers, rapports, comptes rendus, contrats, dossiers, supports de cours, CV, formulaires, documents administratifs, livres blancs ou procédures internes.

Word sert à trois choses principales :

1. **rédiger** du contenu ;
2. **structurer** ce contenu ;
3. **présenter** ce contenu de manière lisible et professionnelle.

La plupart des erreurs viennent d’une confusion entre ces trois niveaux.

Exemple :

* rédiger : écrire un titre ;
* structurer : indiquer que ce texte est un “Titre 1” ;
* présenter : choisir sa taille, sa couleur, son espacement.

Un utilisateur débutant met souvent le titre en gras et en gros manuellement. Un utilisateur avancé applique un style de titre. C’est cette différence qui permet ensuite de générer une table des matières, de modifier tous les titres d’un coup et de rendre le document plus robuste. Les styles Word permettent justement de gagner du temps et d’appliquer une apparence cohérente à l’ensemble d’un document. ([Support Microsoft][2])

## 1.2 La logique de Word

Word fonctionne avec plusieurs niveaux :

| Niveau     | Exemple                             | Rôle                            |
| ---------- | ----------------------------------- | ------------------------------- |
| Caractère  | une lettre, un mot                  | Mise en forme locale            |
| Paragraphe | un bloc de texte terminé par Entrée | Retrait, alignement, espacement |
| Page       | une zone imprimable                 | Marges, orientation             |
| Section    | une partie du document              | Mise en page différente         |
| Document   | fichier complet                     | Structure globale               |

Le point essentiel : dans Word, beaucoup de réglages ne s’appliquent pas seulement au texte visible, mais au paragraphe, à la section ou au document entier.

## 1.3 Les erreurs typiques des débutants

Les erreurs les plus fréquentes sont :

* utiliser plusieurs espaces pour aligner du texte ;
* appuyer plusieurs fois sur Entrée pour créer un espace ;
* écrire les titres sans utiliser les styles ;
* créer une table des matières à la main ;
* déplacer les images sans comprendre l’habillage ;
* insérer des sauts de page au hasard ;
* supprimer un saut de section sans comprendre son rôle ;
* modifier chaque titre manuellement ;
* envoyer un document sans vérifier l’orthographe, l’accessibilité ou la mise en page finale.

## Exercice guidé

Créer un nouveau document intitulé :

**Compte rendu de réunion — Projet Atlas**

Le document doit contenir :

* un titre principal ;
* une date ;
* trois parties ;
* deux paragraphes par partie ;
* une liste à puces ;
* une liste numérotée.

Pour l’instant, ne pas chercher à faire une belle mise en forme. L’objectif est de créer une base de travail.

# Module 3 — Saisie, sélection et correction du texte

## Objectif du module

Savoir saisir, modifier, déplacer et corriger du texte efficacement.

## 3.1 Saisir du texte correctement

Pour écrire dans Word, il suffit de cliquer dans la page et de taper au clavier.

Mais il faut respecter quelques règles :

* une seule espace entre les mots ;
* pas d’espace avant une virgule ou un point ;
* une espace avant les signes doubles en français : deux-points, point-virgule, point d’interrogation, point d’exclamation ;
* éviter les majuscules complètes pour les titres longs ;
* utiliser Entrée uniquement pour terminer un paragraphe ;
* utiliser Maj + Entrée uniquement pour forcer un retour à la ligne sans nouveau paragraphe.

## 3.2 Comprendre la touche Entrée

Dans Word, la touche Entrée ne crée pas seulement un retour visuel. Elle termine un paragraphe.

Cela signifie que le paragraphe possède ses propres réglages :

* alignement ;
* retrait ;
* interligne ;
* espace avant ;
* espace après ;
* style ;
* bordure ;
* puces ;
* numérotation.

Un document mal construit contient souvent trop de paragraphes vides. Ces paragraphes vides perturbent la mise en page.

## 3.3 Sélectionner efficacement

Méthodes utiles :

| Action                        | Méthode                   |
| ----------------------------- | ------------------------- |
| Sélectionner un mot           | double-clic               |
| Sélectionner un paragraphe    | triple-clic               |
| Sélectionner une ligne        | clic dans la marge gauche |
| Sélectionner tout le document | Ctrl + A                  |
| Sélectionner du texte précis  | clic-glissé               |
| Étendre une sélection         | Maj + clic                |
| Sélectionner plusieurs zones  | Ctrl + sélection          |

## 3.4 Copier, couper, coller

Différence importante :

* **Copier** duplique le texte ;
* **Couper** déplace le texte ;
* **Coller** insère le contenu copié ou coupé.

Word propose souvent plusieurs options de collage :

| Option                            | Résultat                    |
| --------------------------------- | --------------------------- |
| Conserver la mise en forme source | garde l’apparence d’origine |
| Fusionner la mise en forme        | adapte au document actuel   |
| Conserver uniquement le texte     | supprime la mise en forme   |

Pour un document propre, il est souvent préférable d’utiliser **Conserver uniquement le texte**, puis de remettre en forme avec les styles du document.

## 3.5 Rechercher et remplacer

La fonction Rechercher permet de trouver rapidement un mot, une expression ou une erreur récurrente.

La fonction Remplacer permet de corriger plusieurs occurrences.

Exemples :

* remplacer “Mme.” par “Mme” ;
* remplacer un double espace par un espace simple ;
* remplacer un ancien nom de projet par un nouveau ;
* homogénéiser une abréviation.

## Exercice guidé

Dans le document :

1. ajouter volontairement plusieurs doubles espaces ;
2. utiliser Rechercher/Remplacer pour les remplacer par une seule espace ;
3. copier un texte depuis une page web ou un autre document ;
4. le coller avec et sans mise en forme ;
5. comparer le résultat.

# Module 4 — Les caractères invisibles

## Objectif du module

Savoir diagnostiquer un document Word en affichant les marques de mise en forme.

## 4.1 Pourquoi afficher les caractères invisibles ?

Les caractères invisibles permettent de voir ce que Word interprète réellement.

Ils révèlent :

* les paragraphes ;
* les espaces ;
* les tabulations ;
* les sauts de page ;
* les sauts de section ;
* les retours à la ligne manuels.

Microsoft recommande d’afficher les marques de mise en forme pour repérer les sauts de section et comprendre où ils commencent et se terminent. ([Support Microsoft][4])

## 4.2 Les principaux symboles

| Symbole         | Signification            |
| --------------- | ------------------------ |
| ¶               | fin de paragraphe        |
| ·               | espace                   |
| →               | tabulation               |
| saut de page    | nouvelle page forcée     |
| saut de section | nouvelle section         |
| ↵               | retour à la ligne manuel |

## 4.3 Pourquoi c’est indispensable

Afficher les caractères invisibles permet de comprendre :

* pourquoi un texte ne s’aligne pas ;
* pourquoi une page blanche apparaît ;
* pourquoi une numérotation change ;
* pourquoi un en-tête disparaît ;
* pourquoi une image se décale ;
* pourquoi un paragraphe conserve une mise en forme étrange.

## Exercice guidé

Dans un document d’exemple :

1. activer les marques de mise en forme ;
2. repérer les paragraphes vides ;
3. supprimer les espaces inutiles ;
4. remplacer les alignements faits avec des espaces par des tabulations ou un tableau ;
5. repérer un saut de page ;
6. repérer un saut de section.

# Module 5 — Mise en forme du texte

## Objectif du module

Savoir mettre en forme du texte proprement sans fragiliser le document.

## 5.1 Mise en forme de caractères

La mise en forme de caractères concerne une partie précise du texte :

* police ;
* taille ;
* gras ;
* italique ;
* souligné ;
* couleur ;
* surlignage ;
* exposant ;
* indice ;
* casse.

Exemples :

* mettre un mot important en gras ;
* écrire H₂O avec un indice ;
* écrire m² avec un exposant ;
* surligner une information à vérifier.

## 5.2 Bonnes pratiques

À faire :

* utiliser le gras pour mettre en valeur ;
* garder peu de polices différentes ;
* éviter les couleurs trop nombreuses ;
* utiliser le souligné surtout pour les liens ;
* garder une hiérarchie claire.

À éviter :

* tout mettre en gras ;
* multiplier les tailles ;
* utiliser trop de couleurs ;
* mélanger plusieurs polices sans raison ;
* utiliser la mise en forme manuelle pour tous les titres.

## 5.3 Mise en forme de paragraphe

La mise en forme de paragraphe concerne tout le bloc de texte :

* alignement ;
* retrait gauche ;
* retrait droit ;
* retrait de première ligne ;
* espacement avant ;
* espacement après ;
* interligne ;
* bordure ;
* trame de fond ;
* puces ;
* numérotation.

## 5.4 Alignements

| Alignement | Usage recommandé                  |
| ---------- | --------------------------------- |
| Gauche     | texte courant                     |
| Centré     | titre court, invitation, affiche  |
| Droite     | date, signature                   |
| Justifié   | documents longs, rapports, livres |

Le texte justifié peut créer de grands espaces entre les mots. Il faut donc l’utiliser avec prudence, surtout sur des colonnes étroites.

## 5.5 Interligne et espacements

Deux notions à distinguer :

* **interligne** : espace entre les lignes d’un même paragraphe ;
* **espacement avant/après** : espace entre les paragraphes.

Bonne pratique : utiliser l’espacement avant/après plutôt que d’appuyer plusieurs fois sur Entrée.

## Exercice guidé

Créer une page contenant :

* un titre ;
* un sous-titre ;
* deux paragraphes ;
* une citation ;
* une liste.

Appliquer une mise en forme cohérente en utilisant :

* gras ;
* italique ;
* alignement ;
* espacement après ;
* interligne ;
* retrait.

# Module 6 — Les styles

## Objectif du module

Comprendre et utiliser les styles, qui sont la base d’un document Word professionnel.

## 6.1 Définition

Un style est un ensemble de réglages de mise en forme enregistré sous un nom.

Un style peut contenir :

* police ;
* taille ;
* couleur ;
* alignement ;
* interligne ;
* espacement ;
* retrait ;
* bordure ;
* niveau hiérarchique.

Microsoft décrit un style comme un ensemble de caractéristiques de mise en forme, par exemple la police, la taille, la couleur, l’alignement, l’interligne, les bordures et l’ombrage. ([Support Microsoft][2])

## 6.2 Pourquoi utiliser les styles ?

Les styles permettent de :

* structurer le document ;
* homogénéiser la présentation ;
* modifier rapidement tout le document ;
* générer une table des matières automatique ;
* naviguer dans le document ;
* améliorer l’accessibilité ;
* éviter les mises en forme incohérentes.

## 6.3 Les styles principaux

| Style        | Usage                          |
| ------------ | ------------------------------ |
| Normal       | texte courant                  |
| Titre        | titre général du document      |
| Titre 1      | grandes parties                |
| Titre 2      | sous-parties                   |
| Titre 3      | sous-sous-parties              |
| Citation     | texte cité                     |
| Légende      | description d’image ou tableau |
| En-tête      | texte dans l’en-tête           |
| Pied de page | texte dans le pied de page     |

## 6.4 Appliquer un style

Procédure :

1. sélectionner le texte ou placer le curseur dans le paragraphe ;
2. aller dans l’onglet **Accueil** ;
3. repérer la galerie **Styles** ;
4. cliquer sur le style souhaité.

## 6.5 Modifier un style

Procédure :

1. aller dans l’onglet **Accueil** ;
2. faire un clic droit sur un style ;
3. choisir **Modifier** ;
4. changer la police, la taille, la couleur, l’espacement ou l’alignement ;
5. valider.

Microsoft indique qu’un style peut être modifié depuis la galerie des styles par clic droit puis **Modifier**, ce qui permet de changer la mise en forme associée au style. ([Support Microsoft][5])

## 6.6 Exemple concret

Si tous les titres de niveau 1 utilisent le style **Titre 1**, il suffit de modifier le style **Titre 1** pour modifier tous les titres de niveau 1 du document.

Sans style, il faudrait modifier chaque titre un par un.

## 6.7 Erreurs fréquentes

| Erreur                                       | Conséquence                           |
| -------------------------------------------- | ------------------------------------- |
| Mettre les titres en gras manuellement       | pas de table des matières automatique |
| Modifier chaque titre séparément             | incohérences                          |
| Utiliser Titre 1 uniquement pour l’apparence | structure incorrecte                  |
| Sauter de Titre 1 à Titre 4                  | hiérarchie confuse                    |
| Copier-coller depuis d’autres documents      | styles parasites                      |

## Exercice guidé

Créer un document avec la structure suivante :

# Projet Atlas

## 1. Présentation

### 1.1 Contexte

### 1.2 Objectifs

## 2. Organisation

### 2.1 Équipe

### 2.2 Planning

## 3. Conclusion

Appliquer :

* **Titre** au titre principal ;
* **Titre 1** aux grandes parties ;
* **Titre 2** aux sous-parties ;
* **Normal** au texte courant.

Modifier ensuite le style **Titre 1** pour que tous les grands titres changent en même temps.

# Module 7 — Les listes à puces et listes numérotées

## Objectif du module

Créer des listes lisibles, hiérarchisées et faciles à modifier.

## 7.1 Liste à puces

Une liste à puces sert à présenter des éléments sans ordre particulier.

Exemple :

* ordinateur ;
* vidéoprojecteur ;
* support de cours ;
* connexion Internet.

## 7.2 Liste numérotée

Une liste numérotée sert à présenter des étapes ou une hiérarchie.

Exemple :

1. ouvrir Word ;
2. créer un document ;
3. saisir le texte ;
4. appliquer les styles ;
5. enregistrer.

## 7.3 Listes multiniveaux

Une liste multiniveau permet de créer une structure hiérarchique.

Exemple :

1. Partie principale
   1.1 Sous-partie
   1.2 Sous-partie
2. Partie principale
   2.1 Sous-partie
   2.2 Sous-partie

## 7.4 Bonnes pratiques

À faire :

* utiliser les boutons de liste ;
* utiliser Tab et Maj + Tab pour changer de niveau ;
* garder un style de puces cohérent ;
* éviter les listes trop longues ;
* utiliser une liste numérotée pour des étapes.

À éviter :

* taper les numéros à la main ;
* mélanger puces et numéros sans logique ;
* créer des indentations avec des espaces ;
* copier-coller des listes depuis plusieurs sources sans nettoyage.

## Exercice guidé

Créer une procédure intitulée :

**Préparer un document professionnel**

Elle doit contenir :

1. une liste numérotée de 6 étapes ;
2. une sous-liste à puces dans l’étape 3 ;
3. une sous-liste numérotée dans l’étape 5.

# Module 8 — Mise en page du document

## Objectif du module

Savoir gérer l’apparence globale des pages.

## 8.1 Marges

Les marges définissent l’espace vide autour du texte.

Elles servent à :

* améliorer la lisibilité ;
* laisser de la place pour l’impression ;
* respecter une charte ;
* équilibrer la page.

Types courants :

* marges normales ;
* marges étroites ;
* marges larges ;
* marges personnalisées.

## 8.2 Orientation

Word propose deux orientations :

| Orientation | Usage                               |
| ----------- | ----------------------------------- |
| Portrait    | courrier, rapport, document courant |
| Paysage     | tableau large, planning, schéma     |

## 8.3 Taille du papier

Formats courants :

* A4 ;
* A3 ;
* Lettre US ;
* enveloppes ;
* formats personnalisés.

## 8.4 Colonnes

Les colonnes servent pour :

* newsletters ;
* articles ;
* brochures ;
* fiches pratiques ;
* documents à lecture rapide.

Attention : les colonnes ne sont pas adaptées à tous les documents. Pour une mise en page très graphique, PowerPoint, Publisher, Canva ou InDesign peuvent parfois être plus adaptés.

## 8.5 Saut de page

Un saut de page force le texte suivant à commencer sur une nouvelle page.

Usage :

* commencer un nouveau chapitre ;
* séparer une page de garde du contenu ;
* éviter qu’un titre important commence en bas de page.

Bonne pratique : utiliser un saut de page plutôt que plusieurs paragraphes vides.

## 8.6 Saut de section

Le saut de section est beaucoup plus puissant que le saut de page.

Il permet de changer la mise en page d’une partie du document :

* orientation différente ;
* marges différentes ;
* colonnes différentes ;
* en-tête différent ;
* pied de page différent ;
* numérotation différente.

Microsoft explique que les sauts de section servent à modifier la mise en page ou la mise en forme d’une partie spécifique d’un document. ([Support Microsoft][6])

## 8.7 Types de sauts de section

| Type          | Usage                                              |
| ------------- | -------------------------------------------------- |
| Page suivante | commence une nouvelle section sur la page suivante |
| Continu       | commence une nouvelle section sur la même page     |
| Page paire    | commence sur la prochaine page paire               |
| Page impaire  | commence sur la prochaine page impaire             |

Microsoft indique qu’un saut de section **Page suivante** démarre la nouvelle section sur la page suivante. ([Support Microsoft][7])

## 8.8 Erreurs fréquentes

| Problème                                        | Cause probable                             |
| ----------------------------------------------- | ------------------------------------------ |
| Page blanche inattendue                         | saut de section mal placé                  |
| Numérotation étrange                            | section différente                         |
| En-tête qui change                              | liaison entre sections                     |
| Orientation paysage qui s’applique partout      | section non isolée                         |
| Suppression d’un saut qui casse la mise en page | saut de section supprimé sans vérification |

Microsoft précise que lorsqu’on supprime un saut de section, Word fusionne les parties avant et après le saut, et la nouvelle section reprend la mise en forme de la section qui suivait. ([Support Microsoft][8])

## Exercice guidé

Créer un document de 4 pages :

* page 1 : page de garde ;
* page 2 : sommaire ;
* page 3 : contenu en portrait ;
* page 4 : tableau en paysage.

Objectif :

* insérer un saut de section avant la page paysage ;
* passer uniquement cette section en paysage ;
* revenir en portrait après.

# Module 9 — En-têtes, pieds de page et numérotation

## Objectif du module

Savoir créer des documents avec des informations répétées et une numérotation propre.

## 9.1 En-tête

L’en-tête est une zone située en haut de la page.

Il peut contenir :

* nom de l’entreprise ;
* titre du document ;
* logo ;
* nom du chapitre ;
* date ;
* version du document.

Microsoft indique que l’insertion d’un en-tête ou d’un pied de page se fait depuis **Insertion > En-tête** ou **Insertion > Pied de page**. ([Support Microsoft][9])

## 9.2 Pied de page

Le pied de page est une zone située en bas de la page.

Il peut contenir :

* numéro de page ;
* confidentialité ;
* nom du fichier ;
* auteur ;
* contact ;
* mention légale.

## 9.3 Numéro de page

Word permet d’insérer automatiquement les numéros de page.

Procédure simple :

1. aller dans **Insertion** ;
2. choisir **Numéro de page** ;
3. choisir la position ;
4. choisir le style.

Microsoft explique que les numéros de page peuvent être ajoutés depuis l’onglet **Insertion**, puis positionnés selon le style choisi. ([Support Microsoft][10])

## 9.4 Numérotation dans l’en-tête ou le pied de page

Procédure :

1. double-cliquer dans l’en-tête ou le pied de page ;
2. aller dans **Numéro de page** ;
3. choisir **Position actuelle** ;
4. sélectionner un style.

Microsoft décrit également l’ajout des numéros de page directement dans un en-tête ou un pied de page. ([Support Microsoft][11])

## 9.5 Ne pas numéroter la page de garde

Méthode courante :

1. ouvrir l’en-tête ou le pied de page ;
2. cocher **Première page différente** ;
3. supprimer le numéro sur la première page si nécessaire ;
4. vérifier que la numérotation continue ensuite.

Microsoft indique que l’option **Première page différente** permet d’éviter l’affichage du numéro sur la première page. ([Support Microsoft][10])

## 9.6 Commencer la numérotation plus tard

Dans un rapport, on peut vouloir :

* ne pas numéroter la page de garde ;
* numéroter le sommaire en chiffres romains ;
* commencer le contenu principal à la page 1.

Cela demande souvent des sauts de section et une numérotation personnalisée.

Microsoft propose une procédure spécifique pour commencer la numérotation plus tard dans un document. ([Support Microsoft][12])

## 9.7 En-têtes différents selon les sections

Un document long peut avoir :

* un en-tête pour l’introduction ;
* un autre pour le chapitre 1 ;
* un autre pour le chapitre 2 ;
* une numérotation différente en annexe.

Pour cela, il faut utiliser les sauts de section et désactiver si besoin l’option **Lier au précédent**.

Microsoft indique que les en-têtes et pieds de page peuvent être configurés différemment selon les sections d’un document. ([Support Microsoft][13])

## 14.2 Suivi des modifications

Le suivi des modifications permet de voir ce qui a été ajouté, supprimé ou modifié.

* travail en équipe ;

On peut afficher :

* toutes les marques ;
* les marques simples ;

# Module 15 — Publipostage

## Objectif du module

## 15.1 Définition

Le publipostage permet de créer plusieurs documents personnalisés à partir d’un document principal et d’une source de données.

* attestations ;
* convocations ;
* certificats ;
* étiquettes ;
* enveloppes ;

Microsoft explique que le publipostage permet de créer un lot de documents personnalisés pour chaque destinataire, en associant un document principal à une source de données comme une liste, une feuille de calcul ou une base de données. ([Support Microsoft][18])

## 15.2 Les trois éléments du publipostage

| Élément            | Rôle                                  |
| ------------------ | ------------------------------------- |
| Document principal | lettre, attestation, e-mail           |
| Source de données  | liste Excel ou base de contacts       |
| Champs de fusion   | zones personnalisées dans le document |

## 15.3 Exemple simple

Source Excel :

| Civilité | Nom    | Prénom | Formation | Date    |
| -------- | ------ | ------ | --------- | ------- |
| Madame   | Martin | Claire | Word      | 12 juin |
| Monsieur | Durand | Hugo   | Excel     | 14 juin |

Document Word :

Madame/Monsieur «Prénom» «Nom»,

Vous êtes inscrit à la formation «Formation» prévue le «Date».

Résultat :

Madame Claire Martin,

Vous êtes inscrite à la formation Word prévue le 12 juin.

## 15.4 Procédure générale

1. préparer la liste Excel ;
2. vérifier les noms de colonnes ;
3. ouvrir Word ;
4. aller dans **Publipostage** ;
5. démarrer la fusion ;
6. sélectionner les destinataires ;
7. insérer les champs de fusion ;
8. prévisualiser les résultats ;
9. finaliser la fusion.

Microsoft documente le publipostage Word avec une feuille Excel comme source de données. ([Support Microsoft][19])

## 15.5 Bonnes pratiques

À faire :

* nettoyer le fichier Excel avant ;
* utiliser des en-têtes de colonnes simples ;
* éviter les cellules fusionnées ;
* vérifier les accents ;
* tester avec 2 ou 3 destinataires ;
* prévisualiser avant l’envoi final.

À éviter :

* utiliser un fichier Excel désordonné ;
* modifier les colonnes après connexion ;
* oublier de vérifier les champs ;
* envoyer sans test ;
* mélanger données personnelles sensibles sans protection.

# Module 17 — Accessibilité

## Objectif du module

Créer des documents lisibles et utilisables par le plus grand nombre.

## 17.1 Pourquoi l’accessibilité est importante

Un document accessible est plus facile à lire pour tout le monde, y compris pour les personnes utilisant un lecteur d’écran ou ayant des difficultés visuelles, cognitives ou motrices.

Microsoft propose des recommandations spécifiques pour rendre les documents Word accessibles aux personnes en situation de handicap. ([Support Microsoft][14])

## 17.2 Principes essentiels

Un document accessible doit avoir :

* des titres structurés ;
* une hiérarchie logique ;
* des textes alternatifs pour les images ;
* des tableaux simples ;
* des lignes d’en-tête dans les tableaux ;
* des liens explicites ;
* un contraste suffisant ;
* une langue correctement définie ;
* une lecture logique.

## 17.3 Titres structurés

Les titres ne doivent pas être simplement gros et gras.

Ils doivent utiliser :

* Titre 1 ;
* Titre 2 ;
* Titre 3.

Cela aide :

* la navigation ;
* les lecteurs d’écran ;
* la table des matières ;
* la compréhension du document.

## 17.4 Texte alternatif

Une image informative doit avoir un texte alternatif.

Exemple :

Image : graphique montrant l’évolution des ventes.

Texte alternatif :

“Graphique montrant une hausse progressive des ventes entre janvier et juin.”

Un tableau accessible doit :

* avoir une ligne d’en-tête ;
* éviter les cellules fusionnées inutiles ;
* rester simple ;
* ne pas servir uniquement à faire de la mise en page ;

## 17.6 Vérificateur d’accessibilité

* le document doit être imprimé ;
* le document est envoyé à un client ;
* le destinataire ne doit pas modifier le contenu ;
* le document doit être archivé.

## 19.4 Impression

Avant impression :

* afficher l’aperçu ;
* vérifier les marges ;
* vérifier l’orientation ;
* vérifier les sauts de page ;
* vérifier les tableaux ;
* imprimer une page test si nécessaire.

# Module 20 — Diagnostic des problèmes fréquents

## Objectif du module

Savoir corriger un document Word qui “part dans tous les sens”.

## 20.1 Le document contient des espaces partout

Symptômes :

* texte mal aligné ;
* colonnes irrégulières ;
* mise en page fragile.

Solution :

* afficher les caractères invisibles ;
* supprimer les espaces multiples ;
* utiliser tabulations, tableaux ou styles.

## 20.2 Les titres ne s’affichent pas dans la table des matières

Causes possibles :

* les titres sont mis en forme manuellement ;
* les styles Titre 1, Titre 2, Titre 3 ne sont pas appliqués ;
* la table n’affiche pas le bon nombre de niveaux.

Solution :

* appliquer les styles de titres ;
* mettre à jour toute la table ;
* vérifier les options de table des matières.

## 20.3 La numérotation recommence toute seule

Causes possibles :

* nouvelle section ;
* liste séparée ;
* mauvais réglage de liste ;
* style mal configuré.

Solution :

* clic droit sur la numérotation ;
* continuer la numérotation ;
* vérifier les listes multiniveaux ;
* vérifier les sauts de section.

## 20.4 Une page blanche apparaît

Causes possibles :

* paragraphes vides ;
* saut de page ;
* saut de section ;
* tableau trop grand ;
* paragraphe suivant gardé avec le précédent.

Solution :

* afficher les marques ;
* repérer les sauts ;
* supprimer les paragraphes inutiles ;
* vérifier les paramètres de paragraphe ;
* vérifier les sections.

## 20.5 L’image bouge

Causes possibles :

* mauvais habillage ;
* image ancrée à un mauvais paragraphe ;
* position libre mal contrôlée.

Solution :

* choisir un habillage adapté ;
* vérifier l’ancre ;
* utiliser “Aligné sur le texte” pour une image stable ;
* utiliser une légende.

## 20.6 L’en-tête change sans raison

Causes possibles :

* plusieurs sections ;
* option “Lier au précédent” active ou désactivée ;
* saut de section supprimé.

Solution :

* afficher les marques ;
* repérer les sections ;
* ouvrir l’en-tête ;
* vérifier “Lier au précédent”.

# Quiz d’évaluation

## Question 1

Pourquoi faut-il utiliser les styles dans Word ?

A. Pour changer uniquement la couleur du texte
B. Pour structurer et harmoniser le document
C. Pour insérer des images
D. Pour imprimer plus vite

Réponse : B

## Question 2

Quelle touche permet d’afficher les caractères invisibles ?

A. Le bouton ¶
B. Le bouton Zoom
C. Le bouton Enregistrer
D. Le bouton Copier

Réponse : A

## Question 3

Quelle est la différence entre un saut de page et un saut de section ?

Réponse attendue :

Un saut de page force le contenu suivant à commencer sur une nouvelle page. Un saut de section crée une nouvelle partie du document qui peut avoir sa propre mise en page, ses marges, son orientation, ses en-têtes, ses pieds de page ou sa numérotation.

## Question 4

Pourquoi une table des matières automatique ne détecte-t-elle pas un titre ?

Réponse attendue :

Parce que le texte a probablement été mis en forme manuellement au lieu d’utiliser un style de titre comme Titre 1, Titre 2 ou Titre 3.

## Question 5

Quel format faut-il privilégier pour envoyer un document final non modifiable ?

A. .docx
B. .dotx
C. .pdf
D. .txt

Réponse : C

## Question 6

À quoi sert le suivi des modifications ?

Réponse attendue :

Il permet de voir les ajouts, suppressions et modifications effectués dans un document, puis de les accepter ou de les refuser.

## Question 7

Pourquoi faut-il éviter d’utiliser plusieurs espaces pour aligner du texte ?

Réponse attendue :

Parce que l’alignement devient instable. Il peut changer selon la police, la taille du texte, l’imprimante ou la largeur de page. Il vaut mieux utiliser les tabulations, les tableaux ou une mise en page adaptée.

## Question 8

À quoi sert un modèle Word ?

Réponse attendue :

Un modèle sert de base réutilisable pour créer plusieurs documents avec la même structure, les mêmes styles, les mêmes en-têtes, les mêmes pieds de page et la même mise en forme.

## Question 9

Que faut-il vérifier avant d’envoyer un document Word ?

Réponse attendue :

Orthographe, commentaires, suivi des modifications, table des matières, numérotation, images, tableaux, accessibilité, mise en page et export PDF.

## Question 10

Pourquoi l’accessibilité est-elle importante dans Word ?

Réponse attendue :

Elle permet de rendre le document plus lisible et utilisable par davantage de personnes, notamment celles qui utilisent des lecteurs d’écran ou qui ont des difficultés visuelles, cognitives ou motrices.

# Conclusion pédagogique

Maîtriser Word ne consiste pas à connaître tous les boutons. La vraie compétence consiste à comprendre la logique du document : texte, paragraphes, styles, sections, champs et finalisation.

Un utilisateur débutant cherche souvent à obtenir rapidement un résultat visuel. Un utilisateur avancé cherche d’abord à créer une structure solide. C’est cette structure qui rend le document facile à corriger, à mettre à jour, à partager, à imprimer et à réutiliser.

La règle essentielle à retenir :

**On ne maîtrise pas Word quand on sait décorer un document. On maîtrise Word quand on sait le structurer proprement.**
