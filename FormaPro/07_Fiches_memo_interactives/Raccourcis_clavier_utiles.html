Insérer un caractère spécial

Alt + code

Maintenir Alt, taper le code sur le pavé numérique, relâcher Alt.

Emoji et symboles

Windows + . ouvre le panneau emoji, symboles et caractères récents.

Word

Insertion > Symbole > Autres symboles, puis choisir police et sous-ensemble.

Copier-coller

Copier un symbole depuis cette fiche, puis coller. Pour garder seulement le texte : Ctrl + Maj + V.

Point important : les codes Alt longs comme Alt + 0176 demandent souvent le pavé numérique. Les résultats peuvent varier selon Windows, clavier, application et police.

Raccourcis Office utiles

Word

  • Ctrl + G : gras.
  • Ctrl + I : italique.
  • Ctrl + U : souligner.
  • Ctrl + E : centrer.
  • Ctrl + J : justifier.
  • Ctrl + L : aligner à gauche.
  • Ctrl + Entrée : saut de page.

Excel

  • F2 : modifier la cellule active.
  • Ctrl + ; : date du jour.
  • Ctrl + Maj + : : heure actuelle.
  • Ctrl + Espace : sélectionner la colonne.
  • Maj + Espace : sélectionner la ligne.
  • Ctrl + 1 : format de cellule.
  • F4 : répéter l'action ou verrouiller une référence dans une formule.

PowerPoint

  • F5 : lancer le diaporama depuis le début.
  • Maj + F5 : lancer depuis la diapositive actuelle.
  • Ctrl + M : nouvelle diapositive.
  • Ctrl + D : dupliquer l'objet ou la diapositive.
  • Ctrl + G : grouper des objets.
  • Ctrl + Maj + G : dissocier.
  • Échap : quitter le diaporama.

Mini-checklist stagiaire

  • Je sais copier, couper, coller et annuler sans chercher dans les menus.
  • Je sais coller en texte brut quand une mise en forme parasite apparaît.
  • Je sais ouvrir le panneau emoji/symboles avec Windows + point.
  • Je sais insérer au moins €, ©, ®, °, É, Ç et ¶.
  • Je vérifie le rendu du symbole dans la police utilisée.
