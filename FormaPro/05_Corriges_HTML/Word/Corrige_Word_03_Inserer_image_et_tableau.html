Corrigé - Insérer une image et un tableau

Corrigé formateur pour l'exercice Word 03.

WordCorrigé formateurCorrection : 12 min

Rappel

Créer une fiche d'accueil avec image et tableau.

Résultat attendu

Titre, texte court, image visible, tableau 3 colonnes avec en-têtes.

Étapes principales

  1. Saisir le titre.
  2. Insérer le texte fourni.
  3. Insérer l'image SVG locale.
  4. Choisir une disposition simple.
  5. Créer le tableau Document / Quantité / Responsable.
  6. Mettre les en-têtes en gras.

Points de contrôle

  • Image non déformée.
  • Tableau complet.
  • En-têtes lisibles.
  • Pas de débordement hors page.

Variantes acceptables

  • Image au-dessus ou sous le texte.
  • Tableau avec fond bleu très clair.
  • Colonnes ajustées différemment.

Correction collective

Faire repérer les problèmes d'habillage d'image, puis montrer le réglage le plus stable.

Erreurs fréquentes

Image trop grande, tableau collé au texte, colonnes trop étroites.

Critères de validation

  • La fiche tient sur une page ou reste bien organisée.
  • Le tableau est compréhensible.
  • L'image illustre sans gêner.
