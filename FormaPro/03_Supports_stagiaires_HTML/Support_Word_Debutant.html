Avant de toucher au document

Créer un fichier propre

  1. Ouvrez Word, puis choisissez Document vierge.
  2. Cliquez sur Fichier, puis Enregistrer sous.
  3. Choisissez le dossier de travail demandé.
  4. Donnez un nom clair, par exemple Courrier_invitation_Martin.docx.
  5. Pendant le travail, utilisez Ctrl + S régulièrement.
Si vous ne retrouvez pas le fichier : ouvrez Fichier > Informations ou Fichier > Ouvrir > Récent. Le chemin du fichier indique le dossier où il est enregistré.

Comprendre la règle de base

Dans Word, on sélectionne d'abord ce que l'on veut modifier, puis on applique une commande. Si rien n'est sélectionné, la commande agit seulement à l'endroit du curseur ou sur le futur texte.

ActionCe qu'il faut sélectionnerCommande utile
Mettre un mot en grasLe motAccueil > G
Centrer un titreLa ligne du titre ou le paragrapheAccueil > Centrer
Changer tout un documentTout le texte avec Ctrl+AAccueil > Police, taille, interligne

Barre d'accès rapide

Ajoutez-y les commandes que vous utilisez tout le temps : Enregistrer, Annuler, Rétablir, Aperçu avant impression. Cela évite de chercher les mêmes boutons à chaque exercice.

Entraînement : repérez Fichier, Accueil, Insertion, Mise en page et Révision. Pour chacun, dites à quoi il sert avant de cliquer.

Saisir, corriger et déplacer du texte

Écrire sans casser la page

  1. Cliquez à l'endroit où le texte doit commencer.
  2. Tapez le texte sans appuyer sur Entrée à la fin de chaque ligne : Word revient à la ligne tout seul.
  3. Appuyez sur Entrée seulement pour créer un nouveau paragraphe.
  4. Utilisez Retour arrière pour effacer à gauche du curseur et Suppr pour effacer à droite.

Copier, couper, coller

BesoinGesteÀ vérifier
Dupliquer un texteSélectionner, Ctrl+C, cliquer à destination, Ctrl+VLe texte existe aux deux endroits.
Déplacer un texteSélectionner, Ctrl+X, cliquer à destination, Ctrl+VLe texte n'existe plus à l'ancien endroit.
Annuler une erreurCtrl+ZLe dernier changement disparaît.
Entraînement 3 minutes : écrivez trois phrases, sélectionnez la deuxième, copiez-la sous la troisième, puis annulez.

Caractères

Les commandes de caractères changent l'apparence des lettres : police, taille, gras, italique, souligné, couleur, surlignage, exposant, indice. Utilisez-les pour hiérarchiser l'information, pas pour décorer chaque ligne.

CommandeEffetBon usage
PoliceChange la forme des lettres.Choisir une police sobre : Calibri, Arial, Verdana.
TailleRend le texte plus grand ou plus petit.Titre plus grand que le corps de texte.
GrasDonne du poids visuel.Titre, mot clé, information prioritaire.
ItaliqueNuance ou distingue.Exemple, citation courte, terme étranger.
SoulignéAjoute un trait sous le texte.À éviter sauf cas précis : cela peut ressembler à un lien.
CouleurClasse ou signale.Deux couleurs maximum dans un document simple.
Exposant / indiceDécale le texte vers le haut ou le bas.m², H₂O, appels de note.

Paragraphes

Les commandes de paragraphe organisent des blocs : alignement, listes, retraits, interligne, espace avant/après. Elles rendent le document plus lisible.

Utiliser les styles et une table des matières

Pourquoi utiliser les styles

Un style applique plusieurs réglages d'un coup. Il évite de refaire à la main la taille, la couleur et l'espacement de chaque titre. Il permet aussi à Word de reconnaître la structure du document.

StyleUsageExemple
NormalTexte courant.Paragraphes du courrier ou de la fiche.
Titre 1Grandes parties.1. Présentation, 2. Expérience.
Titre 2Sous-parties.Missions, Compétences, Formation.

Créer une table des matières automatique

  1. Appliquez Titre 1 aux grandes parties.
  2. Appliquez Titre 2 aux sous-parties.
  3. Placez le curseur au début du document.
  4. Cliquez sur Références > Table des matières.
  5. Choisissez un modèle automatique.
  6. Si les titres changent, cliquez dans la table puis choisissez Mettre à jour.
Si un titre manque dans la table : retournez sur ce titre et appliquez le bon style. Une mise en gras manuelle ne suffit pas.

Préparer la page et le PDF

Marges, orientation et aperçu

  1. Cliquez sur Mise en page ou Disposition.
  2. Choisissez Marges normales si le document est simple.
  3. Gardez Portrait pour courrier et fiche ; utilisez Paysage seulement pour un tableau large.
  4. Ouvrez Fichier > Imprimer pour vérifier l'aperçu.
  5. Si tout est lisible, exportez en PDF avec Fichier > Exporter ou Enregistrer sous > PDF.

Numéroter à partir de la page 2

  1. Insérez d'abord les numéros de page : Insertion > Numéro de page.
  2. Double-cliquez dans l'en-tête ou le pied de page.
  3. Cochez Première page différente pour masquer le numéro de la page de garde.
  4. Sur la page 2, ouvrez Numéro de page > Format des numéros de page.
  5. Choisissez À partir de et tapez 1 si la page 2 doit devenir la page 1.

Sauts de page et sauts de section

Un saut de page force le début d'une nouvelle page. Un saut de section permet de changer une partie du document : orientation, marges, en-tête ou numérotation.

Si une page vide apparaît : activez les marques ¶ dans Accueil. Supprimez les paragraphes vides en trop en fin de document.

Insérer une image et un tableau

Image

  1. Placez le curseur à l'endroit voulu.
  2. Cliquez sur Insertion > Images.
  3. Choisissez l'image, puis validez.
  4. Si l'image gêne le texte, cliquez dessus et choisissez Options de disposition.
  5. Pour débuter, utilisez Aligné sur le texte ou Carré.

Tableau

  1. Cliquez sur Insertion > Tableau.
  2. Choisissez le nombre de colonnes et de lignes.
  3. Saisissez une information par cellule.
  4. Utilisez Création de tableau pour les bordures et Disposition pour ajouter/supprimer une ligne.
  5. Ajustez la largeur si le tableau déborde.

Créer des documents professionnels

Courrier, fiche, CV : même logique

Un document professionnel doit être facile à lire, nommé clairement et exporté en PDF avant l'envoi. La forme sert le contenu : titres nets, peu de couleurs, polices sobres, espaces réguliers.

DocumentStructure conseilléePoint de vigilance
CourrierCoordonnées, date, objet, corps, formule, signature.Objet clair et paragraphes courts.
Fiche de présentationTitre, sections, informations clés, contact.Même style pour tous les titres.
CVIdentité, titre, compétences, expérience, formation, logiciels/langues, centres d'intérêt.Ordre antéchronologique pour les expériences.
Candidature PDFCV + lettre ou message court.Nom de fichier clair : CV_Prenom_Nom.pdf.

QuickParts : blocs réutilisables

Les QuickParts servent à réutiliser un bloc de texte ou un élément récurrent : logo, formule de politesse, signature, mention légale. Chemin : Insertion > Texte > QuickPart.

Contrôle avant de rendre

  • Le fichier est enregistré au bon endroit.
  • Le nom du fichier est clair.
  • Le titre est visible et le texte principal reste lisible.
  • Les styles de titres sont utilisés si le document est long.
  • Les paragraphes ne sont pas collés les uns aux autres.
  • Les images et tableaux ne coupent pas le texte.
  • L'orthographe a été vérifiée dans Révision.
  • L'aperçu avant impression est propre.
  • Le PDF s'ouvre et ressemble au document Word.
