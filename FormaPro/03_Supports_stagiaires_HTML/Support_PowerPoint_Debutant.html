Démarrer une présentation

Créer et enregistrer

  1. Ouvrez PowerPoint.
  2. Choisissez Nouvelle présentation ou un modèle très simple.
  3. Enregistrez immédiatement le fichier avec un nom clair.
  4. Écrivez le sujet dans la première diapositive.
  5. Enregistrez régulièrement avec Ctrl+S.

Construire le message avant la décoration

La règle simple

Une diapositive doit porter une seule idée. Si deux idées se battent sur la même diapositive, créez deux diapositives.

Plan court conseillé

DiapositiveContenuQuestion à se poser
1Titre + contexteDe quoi parle la présentation ?
2Problème ou besoinPourquoi ce sujet existe ?
3Données ou exempleQuelle preuve ou information appuie le message ?
4Solution ou propositionQue faut-il retenir ou décider ?
5ConclusionQuelle est la prochaine action ?
Entraînement : écrivez le titre de 5 diapositives avant de choisir les couleurs ou les images.

Créer des diapositives lisibles

Ajouter une diapositive

  1. Cliquez sur Accueil > Nouvelle diapositive.
  2. Choisissez une disposition simple : Titre et contenu, Deux contenus ou Titre seul.
  3. Écrivez un titre précis.
  4. Ajoutez 3 à 5 points maximum.
  5. Relisez en plein écran : si vous devez plisser les yeux, c'est trop petit ou trop chargé.

Images, formes et mise en page

Insérer une image

  1. Cliquez sur Insertion > Images.
  2. Choisissez une image utile au message.
  3. Redimensionnez depuis un coin pour éviter de la déformer.
  4. Alignez-la avec le texte ou les autres objets.
  5. Supprimez l'image si elle ne rend pas le message plus clair.

Utiliser les formes

Les formes servent à organiser : encadrer une information, montrer une étape, relier deux idées. Gardez peu de couleurs et alignez les objets.

Si tout bouge mal : sélectionnez les objets, puis utilisez Format > Aligner. Dupliquez une diapositive propre au lieu de repartir de zéro.

Insérer depuis Excel

  1. Préparez le tableau ou graphique dans Excel.
  2. Copiez le graphique.
  3. Collez dans PowerPoint.
  4. Redimensionnez sans déformer.
  5. Ajoutez un titre qui dit ce que le graphique montre.

Préparer la prise de parole

Mode diaporama

  1. Cliquez sur Diaporama > À partir du début.
  2. Passez les diapositives avec les flèches du clavier.
  3. Vérifiez que chaque diapositive est lisible sans zoom.
  4. Préparez une phrase d'ouverture et une phrase de conclusion.
  5. Si une diapositive demande trop d'explications, simplifiez-la.

Notes orales

Les notes servent à vous rappeler quoi dire. Elles ne doivent pas être tout le texte de la diapositive.

Contrôle avant de présenter

  • Le fichier est enregistré au bon endroit.
  • Chaque diapositive a un titre clair.
  • Une diapositive = une idée principale.
  • Le texte est lisible en mode diaporama.
  • Les images ne sont pas déformées.
  • Les graphiques ont un titre et des valeurs lisibles.
  • Les transitions restent sobres.
  • La présentation a été testée du début à la fin.
