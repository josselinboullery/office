Créer un classeur propre

  1. Ouvrez Excel et choisissez Nouveau classeur.
  2. Enregistrez immédiatement le fichier dans le bon dossier.
  3. Renommez la feuille avec un nom utile : Budget, Participants, Suivi.
  4. Saisissez d'abord les titres de colonnes.
  5. Enregistrez avec Ctrl+S avant la mise en forme.

Saisir les données correctement

Saisir et valider

  1. Cliquez dans la cellule où l'information doit aller.
  2. Tapez la valeur.
  3. Validez avec Entrée pour descendre ou Tab pour aller à droite.
  4. Pour modifier, recliquez dans la cellule ou utilisez la barre de formule.
  5. Pour supprimer le contenu, sélectionnez la cellule puis appuyez sur Suppr.

Construire un tableau lisible

Ordre conseillé

  1. Ligne 1 : mettez les noms de colonnes.
  2. Une ligne = un élément ou une personne.
  3. Une colonne = un type d'information.
  4. Mettez les en-têtes en gras.
  5. Ajoutez des bordures simples.
  6. Appliquez les formats : nombre, monétaire, pourcentage ou date.

Créer des formules sans se perdre

La méthode en 5 gestes

  1. Cliquez dans la cellule du résultat.
  2. Tapez =.
  3. Cliquez la première cellule utile ou tapez son adresse.
  4. Tapez l'opérateur : +, -, *, /.
  5. Cliquez la deuxième cellule utile, puis validez avec Entrée.

Exemples

BesoinFormuleEn langage naturel
Total ligne=B2*C2Quantité multipliée par prix unitaire.
Somme=SOMME(D2:D8)Additionner tous les montants de D2 à D8.
Moyenne=MOYENNE(D2:D8)Calculer la valeur moyenne de la plage.
Condition SI=SI(C2="Non";"A relancer";"OK")Si C2 contient Non, afficher A relancer ; sinon afficher OK.
Si une formule donne une erreur : cliquez la cellule, lisez la barre de formule, vérifiez les cellules utilisées, puis corrigez une seule chose à la fois.

Recopier, créer des séries et bloquer une référence

Recopie simple

Quand une formule fonctionne sur une ligne, utilisez la poignée de recopie pour l'étendre. Vérifiez ensuite une formule copiée au milieu du tableau.

Références

TypeCe qui se passeExemple
RelativeLa référence change quand on recopie=B2*C2 devient =B3*C3.
AbsolueLa référence reste fixe grâce au signe $=B2*$F$2 garde toujours le taux en F2.
SérieExcel continue une suiteJanvier, Février, Mars ; ou 1, 2, 3.

Préparer l'impression

  1. Passez en Affichage > Mise en page ou ouvrez Fichier > Imprimer.
  2. Choisissez Portrait ou Paysage.
  3. Ajustez les marges si le tableau est trop large.
  4. Utilisez l'échelle seulement si le tableau dépasse légèrement.
  5. Définissez une zone d'impression si seule une partie doit sortir.
  6. Vérifiez que le titre, les colonnes et les totaux sont lisibles.
Si le tableau est coupé : passez en paysage, ajustez à 1 page en largeur, ou réduisez les colonnes inutiles.

Montrer une évolutionCourbeInscriptions par mois.
Montrer une proportion simpleSecteurRépartition d'un budget.

Créer un graphique

  1. Sélectionnez les données utiles avec les en-têtes.
  2. N'incluez pas les totaux si cela fausse le graphique.
  3. Cliquez sur Insertion, puis choisissez le graphique.
  4. Ajoutez un titre clair.
  5. Vérifiez que les axes et la légende sont compréhensibles.

Repérer automatiquement

La mise en forme conditionnelle colore ou signale les cellules selon une règle. Exemple : colorer les écarts négatifs en rouge, afficher des barres de données ou repérer les doublons.

Contrôle avant de rendre

  • Le classeur est enregistré au bon endroit.
  • Les en-têtes sont lisibles.
  • Les montants sont au bon format.
  • Les formules commencent par = et utilisent les bonnes cellules.
  • Les formules recopiées ont été vérifiées sur au moins une ligne.
  • L'aperçu avant impression ne coupe pas le tableau.
  • Le graphique répond à une question claire.
